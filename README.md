

## Setup

Follow these steps to set up the project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd your-repo
   ```

2. **Install dependencies:**
   ```bash
   # For npm
   npm install

   # Or for yarn
   yarn install
   ```

3. **Run the project:**
   ```bash
   # For npm
   npm start

   # Or for yarn
   yarn start
   ```
